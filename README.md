A single-player Pong game for Linux GUI, written in C++ with SDL2. All dependencies are statically linked. No external file dependencies.

## Features
- Player vs CPU Pong
- Ball physics, wall/paddle collisions
- Simple sound effects (no external files)
- Scoring system, win at 10 points
- Statically linked, single executable

## Build Instructions

### Requirements
- Linux
- g++
- SDL2 (static)
- SDL2_mixer (static)

### Build
```
make
```

### Run
```
./Jacob's\ Pong
```

## Controls
- Up/Down Arrow: Move paddle
- Esc: Quit
